### Hook (3 min)
Ask: "How do you learn to ride a bike?"
- Watch others (supervised)
- Try yourself (trial/error - reinforcement)
Say: "ML has similar types of learning!"

### Main Content (18 min)
**Supervised Learning (7 min)**
- Definition: Learning with teacher
- Labeled data
- Two types: Regression vs Classification
- Examples: spam, prices, images
- Quick test: classify problems

**Unsupervised Learning (5 min)**
- Definition: No teacher
- Find patterns
- Clustering example: customers
- When to use

**Reinforcement Learning (6 min)**
- Definition: Trial and error
- Rewards/penalties
- Dog training analogy
- Game playing example
- Self-driving car

### Exercise (2 min)
"Classify these problems:
1. Grouping news articles by topic
2. Predicting student grades
3. Teaching robot to clean"

### Wrap-up (2 min)
- Supervised: labeled data
- Unsupervised: find patterns
- Reinforcement: rewards
- This course: mainly supervised!
