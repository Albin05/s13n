# Lecture Notes: Types of Machine Learning

## Overview Table
| Type | Data Source | Goal | Common Algorithms |
| :--- | :--- | :--- | :--- |
| **Supervised** | Labeled $(X, y)$ | Prediction (Class/Value) | Linear Regression, Decision Trees, SVM |
| **Unsupervised** | Unlabeled $(X)$ | Structure/Pattern Discovery | K-Means, PCA, Apriori |
| **Reinforcement** | Environment Feedback | Strategy/Policy Optimization | Q-Learning, Deep Q-Networks |

## 1. Supervised Learning
* **Regression:** Predicting continuous values (e.g., House Price).
* **Classification:** Predicting categories (e.g., Cat vs. Dog).

## 2. Unsupervised Learning
* **Clustering:** Grouping similar data points (e.g., Customer Segmentation).
* **Dimensionality Reduction:** Simplifying data while keeping information (e.g., compressing images).

## 3. Reinforcement Learning
* **Agent:** The learner.
* **Environment:** The world the agent interacts with.
* **Reward:** Feedback signal (positive or negative).
