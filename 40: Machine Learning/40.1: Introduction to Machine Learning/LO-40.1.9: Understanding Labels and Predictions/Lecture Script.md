### Hook (3 min)
Show photo of a cat
Ask: "What is this?"
Response: "Cat"
Say: "That word 'Cat' is a LABEL! Today we learn about labels."

### Main Content (18 min)
**What are Labels (6 min)**
- Definition: What we predict
- vs Features: What we use to predict
- Symbol: y variable
- Examples: spam/ham, cat/dog, prices

**Labels in Training vs Prediction (7 min)**
- Training: Have labels (learning)
- Prediction: Don't have labels (using)
- Demo: Email spam example with code
- Show: Model learns from labeled data

**Types of Labels (5 min)**
- Binary: Yes/No
- Multi-class: A/B/C/D
- Continuous: Numbers
- Real examples for each

### Exercise (2 min)
"For 'predicting if a student passes':
- What are 3 features?
- What is the label?"

### Wrap-up (2 min)
- Label = target (y)
- Features = inputs (x)
- Need labels to train
- Model predicts labels
