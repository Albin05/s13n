# Question Bank: Real-world Applications of Machine Learning

## Q1 (MCQ)
**Problem:** Which of the following is NOT a typical application of Machine Learning?
A. Predicting stock market prices.
B. Calculating the exact total of a shopping cart given item prices.
C. Diagnosing diseases from medical images.
D. Converting speech to text.

## Q2 (Coding)
**Problem:** You are building a simple "Keyword Spotter" for a customer support bot. Write a function `classify_intent(text)` that returns "Refund" if the word "money" or "refund" is in the text, and "Technical" if "broken" or "error" is in the text. Otherwise, return "General".
**Input:** "I want my money back."
**Output:** "Refund"

## Q3 (NAT)
**Problem:** A fraud detection system reviews 10,000 transactions. It correctly identifies 50 fraudulent transactions but also falsely flags 5 legitimate transactions as fraud. What is the total number of "Positive" predictions (flags raised) by the system?

## Q4 (MSQ)
**Problem:** Which of the following real-world scenarios rely heavily on **Computer Vision**?
A. Self-driving cars detecting stop signs.
B. A smart speaker understanding your voice command.
C. A factory robot identifying defective parts on a conveyor belt.
D. Netflix recommending a movie based on your watch history.

## Q5 (Coding)
**Problem:** Simulate a basic Recommendation System. You have a dictionary `history = {'user1': ['A', 'B'], 'user2': ['B', 'C']}`. Write a function that finds items bought by `user2` that `user1` has NOT bought yet, and returns them as a recommendation for `user1`.
