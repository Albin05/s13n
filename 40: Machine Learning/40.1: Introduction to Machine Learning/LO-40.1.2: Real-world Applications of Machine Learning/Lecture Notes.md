# Real-world Applications of Machine Learning

## 1. Computer Vision (CV)
* **Definition:** Algorithms that interpret visual data (images/video).
* **Applications:**
    * **Face Unlock:** Analyzing facial geometry.
    * **Autonomous Driving:** Detecting pedestrians, signs, and lanes.
    * **Manufacturing:** Quality control (spotting defects on assembly lines).

## 2. Natural Language Processing (NLP)
* **Definition:** Interaction between computers and human language.
* **Applications:**
    * **Chatbots:** Customer support agents.
    * **Translation:** Converting English to Spanish in real-time.
    * **Sentiment Analysis:** Determining if a tweet is happy or angry.

## 3. Recommendation Systems
* **Definition:** Predicting user preference for an item.
* **Applications:**
    * **E-commerce:** "Customers who bought this also bought..."
    * **Streaming:** Personalized playlists (Spotify Discover Weekly).

## 4. Anomaly Detection
* **Definition:** Identifying rare items or events that differ from the norm.
* **Applications:**
    * **Fraud Detection:** Spotting unusual credit card spending.
    * **Cybersecurity:** Detecting network intrusions.
