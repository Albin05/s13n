### Hook (3 min)
Draw two points on graph
Ask: "How would you describe this line mathematically?"
Introduce: y = mx + b
Say: "This simple formula describes ANY straight line!"

### Main Content (18 min)
**Slope (8 min)**
- Definition: rate of change
- Formula: rise/run
- Calculate from points
- Positive/negative/zero
- Interpretation
- Hourly pay example

**Intercept (5 min)**
- Definition: y when x=0
- Starting point
- Visual on graph
- Taxi fare example
- Interpretation

**Complete Examples (5 min)**
- House prices: calculate m and b
- Temperature conversion
- Salary prediction
- Verification

### Exercise (2 min)
"Find equation from:
Point 1: (1, 7)
Point 2: (4, 16)"

### Wrap-up (2 min)
- y = mx + b describes lines
- m = slope = rate
- b = intercept = start
