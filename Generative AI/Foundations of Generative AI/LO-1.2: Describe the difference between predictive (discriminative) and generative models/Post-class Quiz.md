# **Post-Class Quiz**

### **LO-1.2: Predictive vs Generative Models**

---

### **1. What do predictive (discriminative) models primarily learn?**

A) How to generate new samples
B) The decision boundary between classes (P(Y|X))
C) The full data distribution (P(X, Y))
D) Random patterns in training data
**Correct Answer: B**

---

### **2. What is the key capability that generative models have which discriminative models do not?**

A) Predicting labels
B) Reducing overfitting
C) Creating new data similar to the training data
D) Scaling to large datasets
**Correct Answer: C**

---

### **3. Which of the following best describes the core difference between the two model types?**

A) Predictive models create data, generative models predict labels
B) Predictive models learn boundaries; generative models learn distributions
C) Both learn the same thing but use different architectures
D) Generative models are always more accurate
**Correct Answer: B**
