# **Post-Class Quiz (3 Questions)**

**1. Which generative model architecture is primarily used for Large Language Models (LLMs) like ChatGPT?**
A) Convolutional Neural Networks (CNNs)
B) Transformers
C) Generative Adversarial Networks (GANs)
D) Random Forests
E) It was not covered in class
**Correct Answer: B**

---

**2. How do Diffusion Models generate images?**
A) By competing against a discriminator network
B) By predicting the next pixel in a sequence
C) By gradually removing noise from a random static signal
D) By compressing images into a latent space
E) It was not covered in class
**Correct Answer: C**

**3. Which model type involves a "Generator" and a "Discriminator" competing against each other?**
A) VAEs (Variational Autoencoders)
B) Diffusion Models
C) GANs (Generative Adversarial Networks)
D) LLMs (Large Language Models)
