# Generative AI vs Classical Machine Learning

## 1. Definition of Generative AI
- A branch of AI focused on **creating new content**
- Learns probability distribution of data
- Uses models like:
  - Large Language Models (LLMs)
  - Diffusion Models
  - Transformer-based architectures

## 2. Classical Machine Learning Recap

- Predictive in nature  
- Uses algorithms such as:
  - Logistic Regression
  - Random Forest
  - SVMs
  - Gradient Boosting  
- Applications:
  - Classification (spam / not spam)
  - Regression (house prices)
  - Clustering (customer segmentation)

## 3. Key Differences
### Nature of output
- Classical ML → Predicts  
- Generative AI → Creates
<img src="https://coding-platform.s3.amazonaws.com/dev/lms/tickets/203259d8-4d33-403b-9f71-a66164bb7859/zqmpcflrVFJlRNFd.png" alt="image" />


### Input/output relationship
- Classical ML outputs fixed values  
- Generative AI outputs variable, creative sequences  

### Underlying models
- Classical ML uses traditional models  
- Generative AI uses neural architectures with billions of parameters  

## 4. Why this matters
- Generative AI unlocks **automation at scale**
- Enables conversational systems, copilots, agents
- Transforms industries: education, healthcare, business operations
