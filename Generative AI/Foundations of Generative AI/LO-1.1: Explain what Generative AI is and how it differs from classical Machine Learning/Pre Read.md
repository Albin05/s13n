# Pre-Read: Introduction to Generative AI vs Classical ML

## 1. What is Generative AI?
<img src="https://coding-platform.s3.amazonaws.com/dev/lms/tickets/bf25bcea-2da6-4c90-9229-5888270155df/emNFQmHqhzDguHAW.webp" alt="image" />

Generative AI refers to AI systems that can *create* new content—text, images, audio, code, or video—using patterns learned from training data.

Examples:
- ChatGPT generating text
- Midjourney generating images
- Gemini writing code
- Music generation systems

## 2. How is Generative AI different from Classical ML?
| Classical ML | Generative AI |
|--------------|----------------|
| Predicts labels, numbers, or categories | Creates new data/content |
| Uses supervised learning | Uses transformers, diffusion models, LLMs |
| Input → output (fixed mapping) | Input → infinite creative outputs |
| Examples: spam detection, price prediction | Examples: text generation, image synthesis |

## 3. Why is Generative AI important?
- Enables automation of cognitive tasks  
- Powers copilots, chatbots, agents  
- Used in content creation & productivity tools
