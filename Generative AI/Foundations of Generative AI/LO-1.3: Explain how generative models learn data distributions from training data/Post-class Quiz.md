# **Post-Class Quiz (3 Questions)**

### **MCQs**

**1. What is the primary goal of a generative model during training?**
A) To memorize every single data point
B) To learn the underlying probability distribution of the data
C) To classify data into different categories
D) To compress the data into a smaller file
E) It was not covered in class
**Correct Answer: B**

---

**2. In the context of generative AI, what does "sampling" refer to?**
A) Collecting more training data
B) Testing the model's accuracy
C) Generating new data points from the learned distribution
D) Copying an existing image from the dataset
E) It was not covered in class
**Correct Answer: C**

---

**3. If a generative model has perfectly learned the data distribution ($P_{model} \approx P_{data}$), what is the result?**
A) It can predict labels with 100% accuracy
B) It can generate new samples that are indistinguishable from real data
C) It will stop working because it is overfitted
D) It converts all images to text
E) It was not covered in class
**Correct Answer: B**
