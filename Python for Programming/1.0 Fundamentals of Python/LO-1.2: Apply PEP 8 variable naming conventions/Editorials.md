## Problem 5
- `user@domain`: Has @ symbol (invalid)
- `className`: Valid but use `class_name` (snake_case)
- `MAX_value`: Inconsistent (use `MAX_VALUE`)
- `first name`: Has space (invalid)
- `my-var`: Has hyphen (invalid)
