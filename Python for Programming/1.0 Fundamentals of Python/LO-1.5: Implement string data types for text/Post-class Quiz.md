# Post-Class Quiz: Implement String Data Types

## Q1: Valid string?
A) name = "Alice"
B) name = 'Alice'
C) name = `Alice`
D) Both A and B

<details><summary>Answer</summary>
D - Both single and double quotes work
</details>

## Q2: What prints?
```python
print("ha" * 3)
```
A) ha3
B) hahaha
C) ha ha ha

<details><summary>Answer</summary>
B - String repetition
</details>

## Q3: What's wrong?
```python
age = 25
msg = "Age: " + age
```
A) Nothing
B) Can't concatenate str and int
C) Wrong quotes

<details><summary>Answer</summary>
B - Need str(age) or use f-string
</details>

## Q4: What's len("Hello")?
A) 4
B) 5
C) 6

<details><summary>Answer</summary>
B - 5 characters
</details>

## Q5: What prints?
```python
name = "alice"
print(name.upper())
```
A) alice
B) ALICE
C) Alice

<details><summary>Answer</summary>
B - ALICE (all uppercase)
</details>
