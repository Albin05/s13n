# Post-Class Quiz: Implement Float Data Types

## Q1: What's a float?
A) Whole number
B) Number with decimal
C) Text
D) True/False

<details><summary>Answer</summary>
B - Numbers with decimal points
</details>

## Q2: What's the type?
```python
x = 10 / 2
```
A) int  
B) float

<details><summary>Answer</summary>
B - Division always returns float
</details>

## Q3: What prints?
```python
print(0.1 + 0.2)
```
A) 0.3 exactly
B) 0.30000000000000004
C) 0.2
D) Error

<details><summary>Answer</summary>
B - Binary precision limitation
</details>

## Q4: Int + float = ?
A) int
B) float
C) Error

<details><summary>Answer</summary>
B - Result is always float
</details>

## Q5: When to use float?
A) Counting students
B) Product price
C) Year
D) Age

<details><summary>Answer</summary>
B - Prices need decimal precision
</details>
