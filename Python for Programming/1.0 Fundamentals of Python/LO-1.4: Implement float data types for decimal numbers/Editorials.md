# Editorials: Implement Float Data Types

## Problem 1
```python
price = 19.99
height = 5.8
temperature = 72.5

print(type(price))  # <class 'float'>
```

## Problem 2
```python
a = 10.5
b = 3.2

print(a + b)  # 13.7
print(a - b)  # 7.3
print(a * b)  # 33.6
print(a / b)  # 3.28125
```

## Problem 3
```python
price = 50.00
tax_rate = 8.5

total = price * (1 + tax_rate/100)
print(total)  # 54.25
```

## Problem 4
```python
celsius = 25
fahrenheit = celsius * 9/5 + 32
print(fahrenheit)  # 77.0
```

## Problem 5
```python
weight = 70
height = 1.75

bmi = weight / (height ** 2)
bmi_rounded = round(bmi, 2)
print(bmi_rounded)  # 22.86
```
