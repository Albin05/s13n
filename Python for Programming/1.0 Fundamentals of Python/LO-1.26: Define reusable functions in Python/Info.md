# LO-36: Define Functions

## Learning Objective
Students will be able to use function definition and def keyword effectively in Python programming.

## Duration
20-30 minutes

## Key Concepts Covered
1. Understanding function definition and def keyword
2. Syntax and usage
3. Common patterns
4. Best practices and pitfalls

## Prerequisites
- Basic Python syntax
- Control flow structures

## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand function definition and def keyword
- Write correct code using function definition and def keyword
- Apply function definition and def keyword to solve real problems
- Debug common issues
