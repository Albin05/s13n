# Pre-Read: Access List Elements with Indexing

## Accessing List Items

Use index (position number) in square brackets:

```python
fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # "apple"
print(fruits[1])  # "banana"
```

## Important: Starts at 0!

- First item: index 0
- Second item: index 1
- Third item: index 2

## Negative Indexing

```python
print(fruits[-1])  # Last item ("cherry")
print(fruits[-2])  # Second to last ("banana")
```
