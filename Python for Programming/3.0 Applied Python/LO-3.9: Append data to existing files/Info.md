# LO-51: Append to Files

## Learning Objective
Students will be able to append data to existing files without overwriting them.

## Duration
15-20 minutes

## Key Concepts Covered
1. **Append Mode**: Using "a" mode
2. **Preserving Existing Data**: Adding without overwriting
3. **Log Files**: Common use case
4. **File Modes**: Comparing "w" vs "a"

## Prerequisites
- LO-49: Read Text Files
- LO-50: Write Text Files

## Learning Outcomes
By the end of this lesson, students will be able to:
- Open files in append mode
- Add data to existing files
- Understand when to use append vs write
- Create log files and append entries
