# LO-80: Work with Datetime Module

## Learning Objective
Students will be able to use Working with dates and times in Python programs.

## Duration
25-30 minutes

## Key Concepts Covered
1. Working with dates and times
2. datetime module, date, time, timedelta
3. Practical applications
4. Best practices

## Prerequisites
- Basic Python syntax
- Strings
- Basic math operations
- Working with modules

## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand Working with dates and times
- Create and manipulate date/time objects
- Format and parse dates
- Build time-aware applications
