# LO-62: Create Objects

## Learning Objective
Students will be able to use Instantiating objects from classes in Python programs.

## Duration
25-30 minutes

## Key Concepts Covered
1. Instantiating objects from classes
2. object creation, instance creation, calling classes
3. Practical applications
4. Best practices

## Prerequisites
- Basic Python syntax
- Functions
- Previous OOP concepts (LO-61 to LO-61)

## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand Instantiating objects from classes
- Write object-oriented Python code
- Apply OOP principles effectively
- Build reusable, maintainable code
