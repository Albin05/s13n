# LO-73: Create Generator Functions

## Learning Objective
Students will be able to use Writing functions that yield values lazily in Python programs.

## Duration
25-30 minutes

## Key Concepts Covered
1. Writing functions that yield values lazily
2. yield keyword, generator functions, memory efficiency
3. Practical applications
4. Best practices

## Prerequisites
- Basic Python syntax
- Functions
- Iterators and loops
- Understanding of memory concepts

## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand Writing functions that yield values lazily
- Create memory-efficient iterators
- Use generators for large datasets
- Build scalable, performant code
