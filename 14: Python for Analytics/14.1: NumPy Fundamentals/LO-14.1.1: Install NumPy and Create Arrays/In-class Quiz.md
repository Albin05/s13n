## In-class Quiz: Install NumPy and Create Arrays

### Question 1
What is the standard alias used when importing NumPy?

A) `import numpy as num`
B) `import numpy as np`
C) `import numpy as n`
D) `import numpy`

**Correct Answer: B**

---

### Question 2
Which of the following creates a NumPy array from a Python list?

A) `numpy.create([1, 2, 3])`
B) `numpy.list([1, 2, 3])`
C) `numpy.array([1, 2, 3])`
D) `numpy.make_array([1, 2, 3])`

**Correct Answer: C**

---

### Question 3
What is the key difference between Python lists and NumPy arrays regarding data types?

A) Lists are faster than arrays
B) Arrays can only store numbers
C) Arrays must contain homogeneous data (same type)
D) Lists take less memory than arrays

**Correct Answer: C**

---

### Question 4
What will be the output of the following code?
```python
import numpy as np
arr = np.array([1, 2.5, 3])
print(arr.dtype)
```

A) `int64`
B) `float64`
C) `mixed`
D) `Error`

**Correct Answer: B**
*Explanation: NumPy converts all elements to float64 to accommodate the 2.5*

---

### Question 5
Which command is used to install NumPy using pip?

A) `pip get numpy`
B) `pip add numpy`
C) `pip install numpy`
D) `pip download numpy`

**Correct Answer: C**
