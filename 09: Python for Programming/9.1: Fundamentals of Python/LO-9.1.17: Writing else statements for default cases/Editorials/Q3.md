## Problem 3
```python
score = int(input("Enter score: "))

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Grade: {grade}")

if score >= 60:
    print("Result: Passed")
else:
    print("Result: Failed")
```

### Explanation
- Complete if-elif-else chain for grades
- else catches failing scores (< 60)
- Separate if-else for pass/fail status
