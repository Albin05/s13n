## Problem 7: Skip Based on Condition (Medium)

### Solution
```python
i = 0

while i < 20:
    i += 1
    if i > 10 and i % 2 == 0:
        continue
    print(i)
```

```

2
3
4

13
15

19
```

### Explanation
- Print 1-20, but skip even numbers greater than 10
- Increment at start to avoid missing values

- Prints all numbers 1-10, then only odds 11-19
