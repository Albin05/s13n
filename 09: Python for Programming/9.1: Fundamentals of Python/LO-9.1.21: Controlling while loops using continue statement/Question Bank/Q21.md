### **21. Nested Loop Filtering**

Print multiplication table for 1-5, but skip any product that is a multiple of 6.

**Example Output:**
```
1 x 1 = 1
1 x 2 = 2
1 x 3 = 3
1 x 4 = 4
1 x 5 = 5
2 x 1 = 2
2 x 2 = 4
2 x 4 = 8
2 x 5 = 10
...
(skips 2x3=6, 3x2=6, 3x4=12, etc.)
```

**Hint:** Nested loops, calculate product, if `product % 6 == 0`, continue.
