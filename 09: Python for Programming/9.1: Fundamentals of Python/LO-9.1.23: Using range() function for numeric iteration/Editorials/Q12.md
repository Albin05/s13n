## Problem 12: Multiplication Table (Very Hard)

### Solution
```python
n = 7

for i in range(1, 11):
    print(f"{n} × {i} = {n * i}")
```

**Output:**
```
7 × 1 = 7
7 × 2 = 14
7 × 3 = 21
7 × 4 = 28
7 × 5 = 35
7 × 6 = 42
7 × 7 = 49
7 × 8 = 56
7 × 9 = 63
7 × 10 = 70
```

### Explanation
- Range 1 to 10 for multipliers
- Multiply 7 by each number
- Formatted output
- Classic use of range
- Easily adaptable for any number
