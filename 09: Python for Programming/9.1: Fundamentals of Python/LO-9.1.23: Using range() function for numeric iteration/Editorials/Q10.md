## Problem 10: Skip Numbers (Hard)

### Solution
```python
for i in range(0, 20, 3):
    print(i)
```

**Output:**
```
0
3
6
9
12
15
18
```

### Explanation
- Start 0, stop before 20, step 3
- Generates every 3rd number
- Step controls spacing
- 7 numbers total
- Pattern: multiples of 3 less than 20
