## Problem 17: Power Table (Very Hard)

### Solution
```python
base = 2

for exp in range(0, 11):
    result = base ** exp
    print(f"{base}^{exp} = {result}")
```

**Output:**
```
2^0 = 1
2^1 = 2
2^2 = 4
2^3 = 8
2^4 = 16
2^5 = 32
2^6 = 64
2^7 = 128
2^8 = 256
2^9 = 512
2^10 = 1024
```

### Explanation
- Range 0 to 10 for exponents
- Calculate 2 to each power
- Shows exponential growth
- 2^10 = 1024 (kilobyte)
- Range makes pattern clear
