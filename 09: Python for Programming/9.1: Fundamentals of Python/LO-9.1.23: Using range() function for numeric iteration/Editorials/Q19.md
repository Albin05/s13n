## Problem 19: Skip Pattern (Very Hard)

### Solution
```python
for i in range(0, 50, 7):
    print(i, end=" ")
print()
```

**Output:**
```
0 7 14 21 28 35 42 49
```

### Explanation
- Start 0, stop before 50, step 7
- Every 7th number
- Multiples of 7 less than 50
- 8 numbers total
- Custom step creates pattern
