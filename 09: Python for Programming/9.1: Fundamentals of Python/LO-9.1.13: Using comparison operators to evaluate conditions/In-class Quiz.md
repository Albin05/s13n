# Post-Class Quiz: Use Comparison Operators

## Q1: What does `x == 5` do?
A) Assigns 5 to x
B) Checks if x equals 5
C) Changes x to 5

<details><summary>Answer</summary>
B - Checks if x equals 5 (returns True or False)
</details>

## Q2: What's the output?
```python
age = 18
print(age >= 18)
```
A) True
B) False
C) 18

<details><summary>Answer</summary>
A - True (18 is equal to 18, and >= includes equality)
</details>

## Q3: What's the difference?
```python
print(10 > 10)
print(10 >= 10)
```
A) Both True
B) Both False
C) False, then True

<details><summary>Answer</summary>
C - False, then True (> doesn't include equal, >= does)
</details>

## Q4: What type is returned?
```python
result = 5 != 3
print(type(result))
```
A) int
B) bool
C) str

<details><summary>Answer</summary>
B - bool (comparisons always return boolean True/False)
</details>

## Q5: Which is correct?
A) Use `=` to check equality
B) Use `==` to check equality
C) Both work the same

<details><summary>Answer</summary>
B - Use `==` to check equality (`=` is for assignment)
</details>
