## Problem 4: Count Vowels (Easy)

### Solution
```python
text = "hello world"
vowel_count = 0

for char in text:
    if char in "aeiou":
        vowel_count += 1

print(f"Vowels: {vowel_count}")
```

**Output:**
```
Vowels: 3
```

### Explanation
- Iterate through each character
- Check if character is vowel
- Increment counter when found
- Vowels in "hello world": e, o, o = 3
- Simple filtering with for loop
