### **21. Matrix Row Sum**

Calculate sum of each row in a 2D list, but stop if any row sum exceeds 20.

Given matrix:
```python
[[1, 2, 3],
 [4, 5, 6],
 [7, 8, 9]]
```

**Expected Output:**
```
Row 0 sum: 6
Row 1 sum: 15
Row 2 sum: 24 - exceeds limit, stopping
```

**Hint:** Outer loop for rows, calculate sum, break if sum > 20.
