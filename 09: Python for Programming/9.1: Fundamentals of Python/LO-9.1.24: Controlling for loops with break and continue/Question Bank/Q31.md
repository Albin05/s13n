### **31. Validate List**

Check if a list contains only positive numbers. Print "Invalid" and stop if you find a non-positive number.

Test lists:
- List 1: `[5, 10, 15, 20]`
- List 2: `[5, 10, -5, 20]`

**Expected Output:**
```
List 1:
All positive

List 2:
Invalid: -5
```

**Hint:** Use for-else with break when non-positive found.
