### **7. Circle Calculations**

Write a function `circle_info` that takes `radius` as a parameter and prints both circumference and area.

**Expected Output:**
```python
circle_info(5)
# Radius: 5
# Circumference: 31.4159
# Area: 78.53975
```

**Hint:** Circumference = 2πr, Area = πr², use π = 3.14159
