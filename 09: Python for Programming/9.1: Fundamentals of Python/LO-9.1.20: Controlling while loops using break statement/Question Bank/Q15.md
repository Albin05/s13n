### **15. Validate Age Input**

Keep asking for age until user enters a valid age (0-120). Use break to exit when valid.

**Example Interaction:**
```
Enter age (0-120): 150
Invalid! Try again.
Enter age (0-120): -5
Invalid! Try again.
Enter age (0-120): 25
Valid age: 25
```

**Hint:** Use `while True:`, check if `0 <= age <= 120`, break if valid, otherwise continue loop.
