### **12. Menu System**

Create a menu with 4 options. Loop until user chooses to quit (option 4).

Options:
1. Say Hello
2. Say Goodbye
3. Show Time
4. Quit

**Example Interaction:**
```
=== Menu ===
1. Say Hello
2. Say Goodbye
3. Show Time
4. Quit
Choice: 1
Hello!

=== Menu ===
1. Say Hello
2. Say Goodbye
3. Show Time
4. Quit
Choice: 4
Exiting program
```

**Hint:** Use `while True:`, display menu, get choice, use if/elif for options, break on choice 4.
