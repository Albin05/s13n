## Problem 3
```python
bill = 85.00
tip_rate = 0.18
num_people = 4

tip = bill * tip_rate
total = bill + tip
per_person = total / num_people

print(f"Bill: ${bill:.2f}")
print(f"Tip (18%): ${tip:.2f}")           # $15.30
print(f"Total: ${total:.2f}")             # $100.30
print(f"Per person: ${per_person:.2f}")   # $25.08
```
