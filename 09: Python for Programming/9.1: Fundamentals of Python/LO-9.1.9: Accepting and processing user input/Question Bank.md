# Question Bank: Accept User Input

## Problem 1 (Easy)
Get user's name and age, print greeting.

## Problem 2 (Easy)
Ask for two numbers, print their sum.

## Problem 3 (Medium)
Temperature converter:
- Ask for Celsius
- Convert to Fahrenheit
- Display result

## Problem 4 (Medium)
Create a profile:
- Name
- City
- Age
- Favorite color
Print formatted profile.

## Problem 5 (Hard)
Simple quiz:
- Ask "What is 5 + 3?"
- Get answer
- Check if correct
- Print result
