## Problem 1: Age Group Classifier (Easy)

### Solution
```python
age = 25

if age >= 0:
    if age < 13:
        print("Child")
    elif age < 20:
        print("Teenager")
    elif age < 60:
        print("Adult")
    else:
        print("Senior")
else:
    print("Invalid age")
```

**Output:**
```
Adult
```

### Explanation
- Outer `if` checks if age is valid (>= 0)
- Inner conditions classify age groups:
  - 0-12: Child
  - 13-19: Teenager
  - 20-59: Adult
  - 60+: Senior
- If age is negative, prints "Invalid age"
- Nested structure ensures validation before classification
