### **27. Return Sum of List**

Write a function `sum_list` that takes a list of numbers and returns their sum.

**Expected Output:**
```python
total = sum_list([1, 2, 3, 4, 5])
print(total)  # 15

total = sum_list([10, 20, 30])
print(total)  # 60
```

**Hint:** Use loop or built-in `sum()` function.
