Define a function `celsius_to_fahrenheit` that converts Celsius to Fahrenheit and returns the result.

**Expected Output:**
```python

print(f"{f1}째F")  # 32.0째F

f2 = celsius_to_fahrenheit(100)
print(f"{f2}째F")  # 212.0째F
```

**Hint:** Formula: `(celsius * 9/5) + 32`
