### **11. Return from Loop**

Create a function `find_first_even` that takes a list and returns the first even number found.

**Expected Output:**
```python
nums = [1, 3, 5, 8, 9, 10]
result = find_first_even(nums)
print(result)  # 8
```

**Hint:** Loop through list, return immediately when even number found.
