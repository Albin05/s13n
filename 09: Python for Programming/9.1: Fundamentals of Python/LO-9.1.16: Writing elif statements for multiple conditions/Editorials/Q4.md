## Problem 4
```python
score = int(input("Enter score (0-100): "))

if score >= 97:
    print("A+")
elif score >= 93:
    print("A")
elif score >= 90:
    print("A-")
elif score >= 87:
    print("B+")
elif score >= 83:
    print("B")
elif score >= 80:
    print("B-")
elif score >= 77:
    print("C+")
elif score >= 73:
    print("C")
elif score >= 70:
    print("C-")
elif score >= 60:
    print("D")
else:
    print("F")
```

### Explanation
- Check from highest grade to lowest
- Each threshold is carefully ordered
- First True condition wins
- else handles failing grades (below 60)
