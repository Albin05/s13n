## Problem 11: Menu System (Hard)

### Solution
```python
choice = ""

while choice != "4":
    print("\n=== Menu ===")
    print("1. Option A")
    print("2. Option B")
    print("3. Option C")
    print("4. Exit")
    choice = input("Enter choice: ")

    if choice == "1":
        print("You selected Option A")
    elif choice == "2":
        print("You selected Option B")
    elif choice == "3":
        print("You selected Option C")
    elif choice == "4":
        print("Exiting...")
    else:
        print("Invalid choice")

print("Program ended")
```

**Example Output:**
```
=== Menu ===
1. Option A
2. Option B
3. Option C
4. Exit
Enter choice: 1
You selected Option A

=== Menu ===
1. Option A
2. Option B
3. Option C
4. Exit
Enter choice: 4
Exiting...
Program ended
```

### Explanation
- Loop until user chooses to exit
- Display menu each iteration
- Process choice with if/elif
- Invalid input doesn't crash program
- Common pattern for interactive programs
- User controls when to exit
