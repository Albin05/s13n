# LO-68: Use the super() Function

## Learning Objective
Students will be able to use Calling parent class methods from child classes in Python programs.

## Duration
25-30 minutes

## Key Concepts Covered
1. Calling parent class methods from child classes
2. super().__init__(), super().method(), extending functionality
3. Practical applications
4. Best practices

## Prerequisites
- Basic Python syntax
- Functions
- Previous OOP concepts (LO-61 to LO-67)

## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand Calling parent class methods from child classes
- Write object-oriented Python code
- Apply OOP principles effectively
- Build reusable, maintainable code
