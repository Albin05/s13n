### Q1 Solution

```python
# mathutils.py
def add(a, b): return a + b
def subtract(a, b): return a - b
def multiply(a, b): return a * b
def divide(a, b):
    if b == 0: raise ZeroDivisionError
    return a / b

# main.py
from mathutils import add, multiply
print(add(5, 3))       # 8
print(multiply(4, 7))  # 28
```
