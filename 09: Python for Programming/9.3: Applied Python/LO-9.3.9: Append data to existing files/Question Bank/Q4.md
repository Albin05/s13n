## Problem 4 (Medium)

- Function `save_score(name, score)` that appends "name: score" to "scores.txt"
- Function `show_all_scores()` that displays all scores

Example output:
```
Alice: 100
Bob: 95
Charlie: 110
David: 88
Eve: 92
```
