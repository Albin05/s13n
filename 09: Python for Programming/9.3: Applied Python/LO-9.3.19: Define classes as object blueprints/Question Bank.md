## Question Bank: Define Classes as Object Blueprints

---

### Q1: Basic Class Definition (3 minutes, Low Difficulty)

Define a `Car` class with attributes `make`, `model`, and `year`. Create two instances and print their details.

**Key Concepts:** class keyword, __init__, instance attributes

---

### Q2: Class with Methods (3 minutes, Low Difficulty)

Create a `Rectangle` class with `width` and `height` attributes. Add a method `area()` that returns the area and a method `perimeter()` that returns the perimeter.

**Key Concepts:** Instance methods, self parameter

---

### Q3: Class with Default Values (5 minutes, Medium Difficulty)

Create a `BankAccount` class with `owner` and `balance` (default 0). Add methods `deposit(amount)`, `withdraw(amount)`, and `get_balance()`. Withdraw should prevent negative balances.

**Key Concepts:** Default parameters, validation in methods

---

### Q4: Class Representing Data (5 minutes, Medium Difficulty)

Create a `Student` class that stores name, grades (list), and provides methods `add_grade(grade)`, `average()`, and `highest()`. Handle the case where no grades exist.

**Key Concepts:** List attributes, computed properties

---

### Q5: Multiple Interacting Objects (7 minutes, Medium-High Difficulty)

Create a `Playlist` class that stores a name and a list of song titles. Implement `add_song(title)`, `remove_song(title)`, `shuffle()`, and `__len__()` to support `len(playlist)`.

**Key Concepts:** Magic methods, object interaction, random module
