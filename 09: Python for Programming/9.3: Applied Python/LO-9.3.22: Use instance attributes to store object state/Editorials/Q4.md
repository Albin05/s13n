### Q4 Solution

```python
class Record:
    def __init__(self, **kwargs):
        for key, value in kwargs.items():
            setattr(self, key, value)

    def fields(self):
        return list(vars(self).keys())

r = Record(name="Alice", age=25, city="NYC")
print(r.name)      # Alice
print(r.fields())  # ['name', 'age', 'city']
```
