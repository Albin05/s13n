### Q2 Solution

```python
import json
students = [{"name": "Alice", "score": 92}, {"name": "Bob", "score": 85}]
with open("students.json", "w") as f:
    json.dump(students, f, indent=2, sort_keys=True)

with open("students.json") as f:
    for s in json.load(f):
        print(f"{s['name']}: {s['score']}")
```
