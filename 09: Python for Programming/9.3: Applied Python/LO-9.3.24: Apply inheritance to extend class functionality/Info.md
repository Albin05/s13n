# LO-66: Apply Inheritance

## Learning Objective
Students will be able to use Creating subclasses that inherit from parent classes in Python programs.

## Duration
25-30 minutes

## Key Concepts Covered
1. Creating subclasses that inherit from parent classes
2. inheritance syntax, parent class, child class, is-a relationship
3. Practical applications
4. Best practices

## Prerequisites
- Basic Python syntax
- Functions
- Previous OOP concepts (LO-61 to LO-65)

## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand Creating subclasses that inherit from parent classes
- Write object-oriented Python code
- Apply OOP principles effectively
- Build reusable, maintainable code
