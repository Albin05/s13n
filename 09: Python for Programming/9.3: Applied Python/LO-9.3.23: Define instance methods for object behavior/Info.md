# LO-65: Define Instance Methods

## Learning Objective
Students will be able to use Creating methods that operate on instances in Python programs.

## Duration
25-30 minutes

## Key Concepts Covered
1. Creating methods that operate on instances
2. instance methods, self parameter, method calls
3. Practical applications
4. Best practices

## Prerequisites
- Basic Python syntax
- Functions
- Previous OOP concepts (LO-61 to LO-64)

## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand Creating methods that operate on instances
- Write object-oriented Python code
- Apply OOP principles effectively
- Build reusable, maintainable code
