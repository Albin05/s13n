### Q2 Solution: Nested JSON Access

```python
import json
with open("profile.json") as f:
    data = json.load(f)

name = data.get("name", "Unknown")
city = data.get("address", {}).get("city", "Unknown")
theme = data.get("preferences", {}).get("theme", "light")
print(f"{name} from {city}, theme: {theme}")
```
