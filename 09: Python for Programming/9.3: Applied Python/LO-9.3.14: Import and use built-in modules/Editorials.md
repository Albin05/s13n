## Editorials: Import And Use Built-In Modules

---

### Q1 Solution

```python
import math
print(math.sqrt(144))    # 12.0
print(math.ceil(4.3))    # 5
print(math.floor(4.7))   # 4
print(math.pi)           # 3.14159...
print(math.factorial(5)) # 120
```
