## Question Bank: Import and Use Built-in Modules

---

### Q1: Math Module (3 minutes, Low Difficulty)
Use the `math` module to calculate: square root of 144, ceiling of 4.3, floor of 4.7, value of pi, factorial of 5.

**Key Concepts:** math.sqrt, ceil, floor, pi, factorial

---

### Q2: Random Module (3 minutes, Low Difficulty)
Write a program using `random` to: generate random int 1-100, pick random item from list, shuffle a list, generate random float 0-1.

**Key Concepts:** random.randint, choice, shuffle, random

---

### Q3: DateTime Calculator (5 minutes, Medium Difficulty)
Using `datetime`, calculate: days until end of year, your age in days, what day of week a date falls on, add 90 days to today.

**Key Concepts:** datetime, timedelta, strftime

---

### Q4: OS and Sys Info (5 minutes, Medium Difficulty)
Write a script using `os` and `sys` that reports: current directory, Python version, platform, environment variable PATH, list files in current dir.

**Key Concepts:** os.getcwd, sys.version, os.listdir

---

### Q5: Multi-Module Tool (5 minutes, Medium Difficulty)
Build a password generator using `random`, `string`, and `math` modules. Generate passwords of specified length with configurable character sets.

**Key Concepts:** Combining multiple modules
