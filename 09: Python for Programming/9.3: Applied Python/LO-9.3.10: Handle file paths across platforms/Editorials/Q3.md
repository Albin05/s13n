### Q3 Solution: Directory Explorer

```python
from pathlib import Path

def list_files(directory, extension=None):
    d = Path(directory)
    if not d.exists():
        print(f"Directory not found: {d}")
        return []
    if extension:
        files = sorted(d.glob(f"*{extension}"))
    else:
        files = sorted(f for f in d.iterdir() if f.is_file())
    return files
```
