### Q2 Solution

```python
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __eq__(self, other):
        return self.x == other.x and self.y == other.y

p1 = Point(3, 4)
p2 = Point(3, 4)

print(p1 is p2)   # False — different objects in memory
print(p1 == p2)    # True — __eq__ compares values
```
