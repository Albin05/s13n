### Learning Objectives

By the end of this lesson, students will be able to:
1. Add elements using append(), extend(), and insert()
2. Remove elements using remove(), pop(), and clear()
3. Sort and reverse lists in-place
4. Count occurrences and find indices
5. Copy lists using the copy() method
