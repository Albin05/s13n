
# **MCQ Questions**

### **1. Which of the following is a Distributed Version Control System?**

A. SVN
B. Perforce
C. Git
D. Google Docs
**Correct Answer:** C

---

### **2. A developer introduces a bug in the latest update. Which Version Control feature helps restore the last working state?**

A. Version comparison
B. Version rollback
C. UI rendering
D. Code refactoring
**Correct Answer:** B

---

### **3. You want to experiment with a risky feature without affecting the main code. What should you do in Git?**

A. Merge everything to main
B. Create a new branch
C. Delete previous commits
D. Change file names
**Correct Answer:** B

---

### **4. Your teammate accidentally deletes a critical file. How can a VCS help in this situation?**

A. Rewrite from scratch
B. Recover file from commit history
C. Disable collaboration
D. Prevent backups
**Correct Answer:** B
