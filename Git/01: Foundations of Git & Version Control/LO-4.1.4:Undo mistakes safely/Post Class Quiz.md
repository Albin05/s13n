**Q1. What does the command `git restore file.txt` do?**
A) Deletes the file permanently
B) Restores the file to the last committed version
C) Moves the file to staging
D) Modifies the last commit
E) Topic not covered
**Correct Answer: B**

---

**Q2. Which command is used to unstage a file without removing its modifications?**
A) `git commit --amend`
B) `git restore .`
C) `git restore --staged file.txt`
D) `git diff --staged`
E) Topic not covered
**Correct Answer: C**

---

**Q3. When should a developer use `git commit --amend`?**
A) To discard working directory changes
B) To delete the last commit
C) To modify the most recent commit message or include missing changes
D) To reset the repository to an earlier commit
E) Topic not covered
**Correct Answer: C**
