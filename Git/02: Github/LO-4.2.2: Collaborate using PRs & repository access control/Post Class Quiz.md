**Q1. What is the main purpose of cloning a remote repository?**
A) To create a new repository on GitHub
B) To delete the remote repository
C) To create a local copy with full history and a connected remote
D) To push local changes automatically
E) Topic not covered
**Correct Answer: C**

**Q2. What does the command `git push -u origin main` primarily do?**
A) Deletes the remote branch
B) Fetches changes from the remote
C) Sets an upstream relationship for easier future push and pull
D) Creates a new local branch
E) Topic not covered
**Correct Answer: C**

A) It improves Git command performance
