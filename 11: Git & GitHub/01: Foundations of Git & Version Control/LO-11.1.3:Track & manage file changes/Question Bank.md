## **MCQs**

### **Q1. Which Git area contains permanent snapshots of your project?**

A) Working Directory
B) Staging Area
C) Local Repository
D) Trash Bin
E) Topic Not Covered

**Correct Answer:** C

---

### **Q2. What does `git diff` (without flags) compare?**

A) Two commits
B) Staging area vs working directory
C) Staging area vs last commit
D) Working directory vs remote repository
E) Topic Not Covered

**Correct Answer:** B

---

### **Q3. Which command shows files that are staged and ready to commit?**

A) `git add`
B) `git log`
C) `git commit`
D) `git status`
E) Topic Not Covered

**Correct Answer:** D

1. Create a folder named **project-demo**
2. Inside it, create a file **app.js** and write any simple JS code
3. Initialize Git in the folder
4. Stage and commit the file with the message **“Add initial app.js file”**
5. Create another file **utils.js** and write some code
6. Modify **app.js** by adding one more `console.log` line
7. Stage and commit **both files** with the message **“Add utils.js and update app.js”**
