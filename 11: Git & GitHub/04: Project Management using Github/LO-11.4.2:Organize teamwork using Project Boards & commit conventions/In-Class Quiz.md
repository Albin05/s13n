**Q1. What is the primary purpose of using Project Boards in GitHub?**

A) To replace Issues and Milestones completely

B) To visually track the status and flow of issues and pull requests

C) To store commit history permanently

D) To automate code reviews

E) Topic not covered

**Correct Answer: B**
