**Q1. Why should sensitive values like API keys not be hardcoded in source code or workflow files?**

A) They increase application size

B) They reduce application performance

C) They can be exposed publicly and create security risks

D) They prevent workflows from running

E) Topic not covered

**Correct Answer: C**
