**Q1. What does a merge primarily represent in a modern Git workflow?**
A) Copying files from one branch to another
B) Automatically fixing all conflicts
C) Accepting and integrating reviewed changes into the main codebase
D) Restarting development on a new branch
E) Topic not covered
**Correct Answer: C**

---

**Q2. When does Git create a merge conflict?**
A) When branches have different commit messages
B) When two branches modify the same part of a file in incompatible ways
C) When a branch is deleted
D) When pushing code to a remote repository
E) Topic not covered
**Correct Answer: B**

---

**Q3. What is the correct action after resolving merge conflicts in files?**
A) Push changes directly to the remote
B) Leave conflict markers for tracking
C) Stage the resolved files and complete the merge
D) Delete the conflicted branch
E) Topic not covered
**Correct Answer: C**
