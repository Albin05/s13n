### **Q1.** What is the primary goal of **Continuous Integration (CI)**?

A. Automatically deploy code to production
B. Ensure code is tested only at the end of development
C. Frequently integrate code changes and verify them automatically
D. Replace the need for developers to review code

**Correct Answer:** C

---

### **Q2.** Which of the following best describes a **CI/CD pipeline**?

A. A manual checklist followed by developers
B. A sequence of automated steps executed in a defined order
C. A GitHub feature only for deployment
D. A replacement for version control systems

**Correct Answer:** B

---

### **Q3.** What happens if a stage in a CI/CD pipeline fails?

A. The pipeline skips the failed stage and continues
B. The pipeline stops and prevents further steps
C. The pipeline automatically fixes the error
D. The failure is ignored

**Correct Answer:** B
