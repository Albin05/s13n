**Q1. What is the primary role of GitHub Actions in the GitHub workflow?**

A) To manually review pull requests

B) To automatically run commands and validate code when events occur

C) To replace project boards and milestones

D) To store source code securely

E) Topic not covered

**Correct Answer: B**

---

**Q2. In a GitHub Actions workflow, what does a _trigger_ define?**

A) The programming language used in the project

B) The sequence of commands to be executed

C) The event that starts the automation workflow

D) The output format of the console logs

E) Topic not covered

**Correct Answer: C**

---

**Q3. Which statement best explains why GitHub Actions is considered language-independent?**

A) It automatically understands all programming languages

B) It uses the same testing framework for every language

C) The workflow structure remains the same; only the commands change

D) It runs only on JavaScript and Python projects

E) Topic not covered

**Correct Answer: C**
