### **Q1. What is the primary role of GitHub in a Git-based workflow?**

A) Tracking file changes locally
B) Managing branches and commits offline
C) Hosting remote repositories and enabling collaboration
D) Automatically merging all changes

**Correct Answer:** C

---

### **Q2. Which statement best describes the relationship between Git and GitHub?**

A) Git and GitHub are the same tool
B) Git works online, GitHub works offline
C) Git manages code history locally, GitHub manages collaboration remotely
D) GitHub replaces the need for Git

**Correct Answer:** C

---

### **Q3. What does the remote name `origin` represent in Git?**

A) A special Git keyword
B) The default branch name
C) A shortcut name for the primary remote repository
D) A backup copy of the repository

**Correct Answer:** C

### **Q4. Which command creates a local copy of a remote repository and automatically sets up a remote connection?**
