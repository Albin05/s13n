**Q1. What is the primary purpose of using branches in Git?**
A) To increase application performance
B) To isolate different lines of development safely
C) To automatically merge code
D) To replace the main branch
E) Topic not covered
**Correct Answer: B**

---

**Q2. Which command creates a new branch and switches to it in one step?**
A) `git branch feature-login`
B) `git checkout feature-login`
C) `git checkout -b feature-login`
D) `git branch -d feature-login`
E) Topic not covered
**Correct Answer: C**

**Q3. Why is branch cleanup considered a best practice?**
A) It improves Git command speed

**Correct Answer: C**
