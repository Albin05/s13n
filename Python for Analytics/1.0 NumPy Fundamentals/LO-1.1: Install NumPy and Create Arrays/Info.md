# Install NumPy and Create Arrays

## Learning Objective
By the end of this lesson, students will be able to:
- Install NumPy using pip in their Python environment
- Create NumPy arrays from Python lists and tuples
- Understand the difference between Python lists and NumPy arrays
- Create 1D, 2D, and multi-dimensional arrays
- Use array creation functions effectively

## Prerequisites
- Python 3.x installed
- Basic understanding of Python lists and tuples
- Command line/terminal basics

## Key Concepts
- **NumPy**: Numerical Python library for scientific computing
- **ndarray**: N-dimensional array, the core data structure in NumPy
- **Array**: Fixed-size container of homogeneous data types

## Learning Outcomes
Students will gain practical skills in:
1. Installing NumPy via pip
2. Creating arrays from various data sources
3. Understanding array properties and advantages
4. Working with different array dimensions

## Estimated Time
45 minutes

## Difficulty Level
Beginner
