# LO-61: Define Classes

## Learning Objective
Students will be able to use Creating classes with the class keyword in Python programs.

## Duration
25-30 minutes

## Key Concepts Covered
1. Creating classes with the class keyword
2. class definition, naming conventions, class body
3. Practical applications
4. Best practices

## Prerequisites
- Basic Python syntax
- Functions


## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand Creating classes with the class keyword
- Write object-oriented Python code
- Apply OOP principles effectively
- Build reusable, maintainable code
