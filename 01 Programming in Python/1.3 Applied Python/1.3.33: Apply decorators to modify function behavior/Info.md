# LO-75: Apply Decorators

## Learning Objective
Students will be able to use Using decorators to modify function behavior in Python programs.

## Duration
25-30 minutes

## Key Concepts Covered
1. Using decorators to modify function behavior
2. @ syntax, wrapper functions, function modification
3. Practical applications
4. Best practices

## Prerequisites
- Basic Python syntax
- Functions
- Higher-order functions
- Understanding of closures

## Learning Outcomes
By the end of this lesson, students will be able to:
- Understand Using decorators to modify function behavior
- Create and apply decorators
- Use built-in decorators
- Build reusable, modular code
