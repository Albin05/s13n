# LO-22: Write For Loops

## Learning Objective
Students will be able to use for loops effectively in Python programs.

## Duration
20-25 minutes

## Key Concepts Covered
1. Syntax and structure
2. Practical applications
3. Common patterns
4. Best practices

## Prerequisites
- Previous LOs on control flow

## Learning Outcomes
By the end of this lesson, students will be able to:
- Write correct for loops
- Apply them to solve problems
- Debug common issues
- Understand when to use for loops
