# Post-Class Quiz: Convert Between Data Types

## Q1: Convert "25" to int?
A) int("25")
B) integer("25")
C) toInt("25")

<details><summary>Answer</summary>
A - int("25")
</details>

## Q2: What's int(3.9)?
A) 3
B) 4
C) 3.9

<details><summary>Answer</summary>
A - 3 (truncates, doesn't round)
</details>

## Q3: User input type?
```python
x = input("Number: ")
```
A) int
B) str
C) float

<details><summary>Answer</summary>
B - input() always returns string
</details>

## Q4: Fix this:
```python
age = "25"
result = age + 5
```
A) int(age) + 5
B) age + str(5)
C) Both work

<details><summary>Answer</summary>
A - Convert string to int for math
</details>

## Q5: What's bool(0)?
A) True
B) False
C) 0

<details><summary>Answer</summary>
B - 0 is False, non-zero is True
</details>
