# Lecture Script: LO-6 Implement Boolean Data Types

## [0:00-0:02] Hook
**Say**: "Is the light on or off? Is the door open or closed? Life is full of yes/no questions - that's what booleans are for!"

## [0:02-0:06] Boolean Basics (4 min)
- Two values: True/False
- Must be capitalized
- Show examples

## [0:06-0:10] Practice (4 min)
Create booleans for:
- Is it sunny? (True)
- Do you have homework? (False)
- Is class over? (False)
